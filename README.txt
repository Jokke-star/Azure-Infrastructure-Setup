This PowerShell script automates the deployment of a Windows Server 2019 Virtual Machine in Microsoft Azure. It creates all necessary infrastructure components, including:

- Resource Group  
- Virtual Network and Subnet  
- Network Security Group with RDP access  
- Public IP Address  
- Network Interface  
- Windows Server 2019 VM  

The script uses the [Az PowerShell module](https://docs.microsoft.com/en-us/powershell/azure/).\
